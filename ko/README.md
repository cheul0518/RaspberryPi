# 라즈베리 파이

이 저장소는 라즈베리 파이 활용을 위한 프로젝트입니다.

## 라즈베리 파이 v. 아두이노

| 항목 | 라즈베리 파이 | 아두이노 |
|-----|-----|-----|
| 종류            | 소형 컴퓨터           | 마이크로 컨트롤러 |
| 운영체제        | Linux 기반 OS         | 없음              |
| 프로그래밍 언어 | Python, C/C++ 등      | C/C++             |
| 멀티태스킹      | 가능                  | 불가능             |
| 구성요소        | CPU, RAM(고), 저장장치(SD 등), I/O | CPU, RAM(소), Flash|
